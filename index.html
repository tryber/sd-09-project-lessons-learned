Lições aprendidas do Gabriel Oliva com a T9

Lições Aprendidas

  1. Não deixar o Arlesson assumir o Host da reunião do zoom
  2. git commit é diferente de got commit
  3. A partir de hoje, todo novo repositório colocar a branch principal com o nome main
  4. Deixar nosso HTML semântico pode ser um dividor de águas na vida de pessoas com algum tipo de deficiencia visual
  5. Não arrumar a cortina do quarto faltando 1 minuto para começar a aula
  6. Não mandar um @here no slack da turma e ir arrumar a cortina do quarto
  7. Confirmar a troca de Host antes de executar tal ação
  8. Não deixar um copo de água próximo ao meu mouse na hora da aula ao vivo
  9. O poder pode subir na cabeça do Arlesson das pessoas quando o Host da reunião é dado
  10. Arrumar CSS do site é igual arrumar cortina do meu quarto, uma hora cai.
