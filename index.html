As duas primeiras semanas na Trybe

A exatas duas semanas do início do curso, o aprendizado ultrapassa de longe o que esperava até aqui.

O primeiro passo foi aprender sobre bash, kernell e os principais comandos do terminal, já que é preferível que utilizemos essa ferramenta pra manipular e criar diretórios.

Em seguida, aprendemos sobre Git e GitHub. É um sistema de controle de versão. Uma ferramenta indispensável, que garante que a gente não perca as evoluções do projeto e permite que mais pessoas trabalhem em um mesmo projeto sem que seja uma catástrofe total e ninguém perca o trabalho de ninguém. Aprendemos os principais comandos, como criar branchs, criar repositórios, subir as alterações para o repositório remoto que é o GitHub, puxar alterações do repositório remoto para o repositório local ( nossa máquina).

O terceiro passo foi aprender sobre o HTML, que é a estrutura das páginas web como conhecemos. É uma página estática, por isso, uma página só com HTML lembra muito aqueles sites e blogs que visitávamos no início do século, quando a internet ainda era discada, lembra ?

Ainda sobre HTML descobrimos uma coisa chamada HTML semântico, algo que eu nem sonhava que podia existir. É uma forma mais clara de usar as tags HTML, o que é fundamental para a acessibilidade de pessoas que não enxergam, os leitores de tela se guiam pelas tags e o uso correto delas é primordial para o bom funcionamento desses leitores.

A quarta coisa que aprendemos foi o CSS, que é uma forma de estilizar as páginas HTML, editar toda a aparência da página, deixando ela sem aquela cara de página web do início do século. Aprendemos até a colocar imagem no plano de fundo do site, arredondar bordas, fazer o objeto ( seja texto ou imagem) flutuar, colocar ele da forma que a gente bem entender e quiser. Apesar de serem fundamentais no desenvolvimento web CSS e HTML não são linguagens de programação, são linguagens de marcação.
