Momentos de aprendizado

O dia em que eu aprendi a usar o Github


E então conheci a ferramenta que me acompanhá por todo o resto de minha vida como programador...


Custei a entender o propósito do GitHub em meu primeiro contato, achei complicado e até desnecessário. Eu poderia muito bem guardar meus trabalhos no Drive (risos). Na verdade eu sentia o github uma coisa muito distante, coisa que só Devs experientes usavam em seus projetos open-source para salvar vidas de usuários aflitos com o linux (eu).

Mas graças a comunidade da Trybe eu tive a oportunidade de conhecer o passo a passo de como usar o GitHub na prática. E cá estou realizando meu primeiro projeto tranquilamente, sem perder uma vírgula do meu código, com muitos commits, pushs e reviews.

O git foi complicado no começo em questão de organização do espaço de trabalho. Mas como todo conhecimento novo, a prática fez a coisa começar a fazer sentido. E foi assim que segui aprendendo as outras lições que a Trybe ofereceu: com muita prática e desafio!
